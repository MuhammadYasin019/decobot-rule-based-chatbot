# 🤖 DecoBot: Rule-Based AI Chatbot

A clean, responsive, and interactive rule-based chatbot built with Python and Streamlit. This application bypasses standard chat UI blocks to offer stylized messaging bubbles alongside real-time typing dynamics.

Developed as **Project 1** during the **DecodeLabs AI Track Internship (Batch 2026)**.

---

## 🚀 Features

* **Dynamic Time-Based Greetings:** Automatically welcomes the user with contextual greetings (Morning, Afternoon, Evening, Night) depending on the system clock.
* **Custom Chat UI:** Uses injected HTML/CSS to render distinct, eye-catching chat bubbles (blue for user, green for assistant).
* **Typing Animation Effect:** Features a smooth visual text streaming layout using character-by-character delays to mimic human interaction.
* **Exact Matching & Text Sanitization:** Strips trailing punctuation, clears excess whitespace, and handles case-insensitive queries to match defined rules.
* **Robust Fallback Engine:** Rotates through unique fallback strings to keep the conversation natural when an input falls outside the rule base.
